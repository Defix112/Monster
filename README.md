# Новогодний Калькулятор - PWA

Красивый новогодний калькулятор с поддержкой офлайн работы (PWA).

## Установка на телефон

### Android (Chrome):
1. Откройте сайт в браузере Chrome
2. Нажмите на меню (три точки)
3. Выберите "Добавить на главный экран" или "Установить приложение"
4. Приложение будет установлено и будет работать без интернета

### iOS (Safari):
1. Откройте сайт в Safari
2. Нажмите кнопку "Поделиться"
3. Выберите "На экран «Домой»"
4. Приложение будет добавлено на главный экран

## Быстрый старт

1. **Скачайте архив** `Calculator-PWA.zip` (уже создан в папке проекта)
2. **Распакуйте** все файлы в одну папку
3. **Создайте иконки**: откройте `create-icons.html` в браузере - иконки автоматически скачаются
4. **Поместите иконки** (`icon-192.png` и `icon-512.png`) в ту же папку с остальными файлами
5. **Загрузите на веб-сервер** или используйте локальный сервер для тестирования

## Создание иконок

Для работы PWA нужны иконки. Есть два способа их создать:

### Способ 1: Через браузер (рекомендуется)
1. Откройте файл `create-icons.html` в браузере
2. Иконки автоматически скачаются (`icon-192.png` и `icon-512.png`)
3. Поместите их в папку с остальными файлами

### Способ 2: Через Node.js
```bash
npm install canvas
node generate-icons.js
```

## Файлы проекта

- `index.html` - главная страница
- `style.css` - стили
- `script.js` - логика калькулятора
- `manifest.json` - манифест PWA
- `service-worker.js` - сервис-воркер для офлайн работы
- `icon-192.png` - иконка 192x192 (создать через create-icons.html)
- `icon-512.png` - иконка 512x512 (создать через create-icons.html)

## Использование

1. Откройте `index.html` в браузере
2. Создайте иконки через `create-icons.html`
3. Загрузите все файлы на веб-сервер (для PWA нужен HTTPS или localhost)
4. Откройте сайт на телефоне и установите приложение

## Для локального тестирования

Запустите локальный сервер:
```bash
# Python 3
python -m http.server 8000

# Node.js (http-server)
npx http-server -p 8000
```

Затем откройте http://localhost:8000
